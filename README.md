# To-Do List
A practical tool that helps you not to forget your tasks

## Technologies
- HTML5
- CSS3
- SASS
- JavaScript

## Features 
- Delete task
- Check your task when it is done 

## Challenges
- Storing the tasks and its status in the browser's localStorage 
- Adding micro animations to the web project
